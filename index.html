OCR With JavaScript

OCR means Optical Character Recognition. It is a technology that reads text from an image, photo, screenshot, or camera frame.

Image To Text Demo Camera To Text Demo

What This Project Does

This website uses JavaScript and Tesseract.js to recognize English and Khmer text directly inside the browser. The text does not need to be typed manually.

First, the browser gets an image. Next, Tesseract.js studies the image and tries to find letters. Finally, it returns the letters as normal text that can be copied or edited.
